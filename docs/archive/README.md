# Archived Docs

This folder keeps historical planning and research documents that are no longer the current operating
path. They are preserved for context and traceability, not for day-to-day live-test instructions.

- `legacy/` — pre-official-runtime daemon plans, early progress logs, and old local STT/TTS brain
  research.
- `research/` — superseded or paused implementation research, including the old LiveKit backend plan.

Current docs remain at the top level of `docs/`, especially `runbook.md`, `todo-official-runtime.md`,
`plan-official-runtime-refactor.md`, `live-test-log.md`, and active backend/context research.
